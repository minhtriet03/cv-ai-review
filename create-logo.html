Để lưu logo này:

  1. Chụp màn hình logo (sử dụng công cụ chụp màn hình hệ thống)
  2. Chỉnh sửa và lưu thành các file sau:
    • favicon.ico - 16x16, 32x32, 48x48
    • logo192.png - 192x192
    • logo512.png - 512x512
    • new-logo.png - kích thước tùy chọn
  3. Lưu các file trong thư mục: frontend/public/images/
